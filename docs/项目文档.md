# 在线聊天室 - 项目文档

## 1. 项目简介

本项目是一个基于 Web 的在线聊天室系统，支持用户注册、登录和实时文字聊天。所有用户在同一群组内发送和接收消息，消息包含发送人昵称和发送时间。

## 2. 技术选型与理由

| 技术 | 选型理由 |
| --- | --- |
| 后端框架：Flask | 轻量级、快速开发、Python生态完善 |
| 数据库：MySQL 8.0 | 稳定可靠，满足数据持久化需求 |
| 实时通信：WebSocket | 支持双向实时通信，消息即时推送 |
| 密码加密：bcrypt | 安全哈希算法，加盐处理 |
| 前端：HTML/CSS/JS | 原生三件套，无需额外框架 |

## 3. 数据库表结构设计

### 3.1 用户表 (users)

| 字段名 | 类型 | 约束 | 说明 |
| --- | --- | --- | --- |
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 用户唯一ID |
| username | VARCHAR(50) | UNIQUE, NOT NULL | 登录用户名 |
| password_hash | VARCHAR(255) | NOT NULL | bcrypt加密后的密码 |
| nickname | VARCHAR(50) | NOT NULL | 显示昵称 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 注册时间 |

### 3.2 消息表 (messages)

| 字段名 | 类型 | 约束 | 说明 |
| --- | --- | --- | --- |
| id | INT | PRIMARY KEY, AUTO_INCREMENT | 消息唯一ID |
| user_id | INT | NOT NULL, FOREIGN KEY | 发送用户ID |
| nickname | VARCHAR(50) | NOT NULL | 发送人昵称 |
| content | TEXT | NOT NULL | 消息内容 |
| created_at | TIMESTAMP | DEFAULT CURRENT_TIMESTAMP | 发送时间 |

### 3.3 建表 SQL

| 序号 | SQL语句 |
| --- | --- |
| 1 | CREATE DATABASE chatroom; |
| 2 | USE chatroom; |
| 3 | CREATE TABLE users ( id INT AUTO_INCREMENT PRIMARY KEY, username VARCHAR(50) UNIQUE NOT NULL, password_hash VARCHAR(255) NOT NULL, nickname VARCHAR(50) NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); |
| 4 | CREATE TABLE messages ( id INT AUTO_INCREMENT PRIMARY KEY, user_id INT NOT NULL, nickname VARCHAR(50) NOT NULL, content TEXT NOT NULL, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP, FOREIGN KEY (user_id) REFERENCES users(id) ); |

## 4. API 接口列表

### 4.1 注册接口

| 项目 | 内容 |
| --- | --- |
| URL | /api/register |
| 方法 | POST |
| 请求体 | {"username": "alice", "password": "123456", "nickname": "爱丽丝"} |
| 响应(成功) | {"success": true, "message": "注册成功"} |
| 响应(失败) | {"success": false, "message": "用户名已存在"} |

### 4.2 登录接口

| 项目 | 内容 |
| --- | --- |
| URL | /api/login |
| 方法 | POST |
| 请求体 | {"username": "alice", "password": "123456"} |
| 响应(成功) | {"success": true, "message": "登录成功", "nickname": "爱丽丝"} |
| 响应(失败) | {"success": false, "message": "用户名或密码错误"} |

### 4.3 获取历史消息

| 项目 | 内容 |
| --- | --- |
| URL | /api/messages |
| 方法 | GET |
| 响应(成功) | {"success": true, "messages": [{"nickname": "爱丽丝", "content": "大家好", "created_at": "20:47:31"}]} |

### 4.4 WebSocket 消息发送

| 项目 | 内容 |
| --- | --- |
| 事件名 | send_message |
| 数据格式 | {"content": "大家好"} |
| 广播事件 | new_message |
| 广播数据 | {"nickname": "爱丽丝", "content": "大家好", "created_at": "20:47:31"} |

## 5. 项目运行步骤

### 5.1 环境要求

| 项目 | 要求 |
| --- | --- |
| Python | 3.10 或更高版本 |
| MySQL | 8.0 或更高版本 |
| 浏览器 | Chrome / Edge  |

### 5.2 安装步骤

| 步骤 | 操作 |
| --- | --- |
| 1 | 安装 MySQL 8.0，设置 root 密码 |
| 2 | 连接 MySQL，执行建表 SQL |
| 3 | 修改 app.py 中的数据库密码 |
| 4 | 安装依赖：pip install flask flask-socketio mysql-connector-python bcrypt eventlet |
| 5 | 运行：python app.py |
| 6 | 访问：http://localhost:5000 |
